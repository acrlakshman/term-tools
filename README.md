term-tools
==========

Wenzel's terminal setup. Contains nice default settings for

* vim
* tmux
* zsh
* git
* octave
* xterm
* karabiner (OSX)

How to install
--------------

* Clone this repository:
```
git clone https://github.com/wjakob/term-tools.git ~/.term-tools
cd ~/.term-tools
./install.sh
```
